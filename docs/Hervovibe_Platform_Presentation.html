Admin Panel Features

What the Admin Can Do

The master admin has a complete control panel to manage the entire MLM network from one place.

👥

Member Management

Add, view, edit, and manage all distributors in the network. Every member gets a unique ID, sponsor link, and login credentials.

  • Custom Member ID — admin can set (e.g. HV-0001) or let system auto-generate
  • Member ID is editable anytime from the Edit Member screen
  • All member dropdowns show [ID] — Name format with search by ID or name
  • Auto-generated secure temporary password on member creation
  • Filter members by rank, status, and ID color (GREEN / YELLOW / RED / BLACK)
  • View any member's full profile, 8-level tree, earnings, and sales history
💰

Sales Entry & Invoice Upload

Record every product sale in the system. Each sale automatically triggers PI, BI, and business commission calculations up the entire sponsor chain — instantly.

  • Select member (searchable by ID or name) + select products + enter quantities
  • Upload invoice screenshot / photo (stored securely)
  • System auto-calculates total value, PI earned, BI earned, and all commissions
  • Commission and incentives instantly credited up the full chain
  • Delete a sale if entered by mistake
📦

Product Catalog Management

Maintain the Hervovibe product catalog with per-product incentive rates. Different products can have different PI% and BI%.

  • Add product: Name + MRP + PI% (Product Incentive rate) + BI% (Business Incentive rate)
  • Default: PI = 10% of MRP · BI = 1% of MRP at Level 1
  • Live preview while adding: shows exact ₹ amount per unit for both PI and BI chain
  • Products appear in sales entry dropdown instantly after adding
  • Edit rates anytime — future sales use updated rates
🏆

Automatic Rank Engine

The system automatically checks and upgrades member ranks — no manual work needed. Ranks are permanent once achieved.

  • Admin runs rank check for any month with one click
  • Checks: personal purchase ≥ ₹1,260 AND minimum active (GREEN) team size
  • Promotes eligible members to next rank automatically
  • Salary withheld if inactive that month — but rank is never removed
  • Full rank change history maintained per member
💸

Payout Management — 4 Types

The payout system handles all 4 types of payments separately. Monthly payouts are released at month-end. BI is released whenever the admin decides.

Monthly — released every month

Monthly Payouts

  • Rank Salary — based on current rank (₹500 Bronze → ₹1 Lakh Centennial)
  • Business Commission — % of sale amount, rank-gated up 7 upline levels
  • PI (Product Incentive) — PI% of MRP, seller keeps full amount, upline splits equally

Admin clicks "Generate Payouts" → previews all 3 components for every member → releases with one click

Admin-scheduled — 3, 6 or 12 months

BI Release (Business Incentive)

  • BI% of MRP — total BI equally split among all upline members (same rule as PI)
  • Accumulates in member profile automatically from every sale
  • Admin selects a date range (e.g. Jan–Mar 2026) + labels it (e.g. "Q1 2026")
  • System shows total accumulated BI per member for that period
  • Admin releases with one click — full history maintained
📊

Incentives Dashboard

View auto-calculated PI and BI commission records, configure the BI base rate, and manage PI rate settings.

  • PI Tab — view all PI commissions earned per member per month (auto from sales)
  • BI Tab — configure BI base rate (default 1%), view accumulated BI per member per period
  • PI Rate Tab — set monthly ₹/point conversion rate if using point-based system
  • All Commissions Tab — full breakdown of Business + PI + BI per member for any month
📈

Admin Dashboard & Reports

A birds-eye view of the entire network — all key numbers at a glance.

  • Total members, active members, RED / BLACK members count
  • Total network sales this month
  • Total commissions & incentives distributed
  • Member list filterable by rank, status, ID color
  • Export payout data to CSV

2. PI — Product Incentive (Monthly)

PI% of MRP per unit sold (default 10%, configurable per product). Seller keeps the full amount. All upline members share an equal split of the same PI amount.

SellerFull PI amount
All Upline (equally)PI ÷ no. of upline

Example: MRP ₹600 × 10% = ₹60 PI. Seller gets ₹60. If 6 upline members → each gets ₹10.

3. BI — Business Incentive (Admin-scheduled)

BI% of MRP per unit sold (default 1%, configurable per product). Same rule as PI — all upline members share an equal split. Accumulates in member profile, released by admin.

All Upline (equally)BI ÷ no. of upline

Example: MRP ₹600 × 1% = ₹6 BI. If 6 upline members → each gets ₹1. Admin releases quarterly, half-yearly, or yearly.
